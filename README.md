# Security Report Generator


---

## Sobre o projeto

Script Python que gera automaticamente um relatório HTML de incidentes de segurança com KPIs de SOC formatado, visual e pronto para ser aberto no navegador, tudo criado com Python, Pandas, HTML e CSS.

Esse script foi criado com o intuito de explorar um complemento na etapa de Análise de Dados, e Python é a linguagem que estou aprendendo como principal, então esse projeto foi uma forma de juntar o que aprendi em análise de dados com o que estou desenvolvendo em programação e ainda enriquecer o meu portfólio com algo que mostra até onde vai o meu conhecimento até agora.

---

## O que o relatório contém

 KPIs principais: total de incidentes, severidade média, tempo de resposta
 Tabela detalhada de incidentes por tipo e status
 Resumo por categoria de evento
 Layout visual limpo gerado diretamente pelo script

---

## Tecnologias Utilizadas

| Ferramentas utilizadas:

 Python | Linguagem principal |
 Pandas | Processamento e análise dos dados |
 HTML + CSS | Estrutura e estilo do relatório gerado |

---

## Como executar

Pré-requisitos: Python 3.x instalado.

```bash
# Instalar dependências
pip install pandas

# Rodar o script
python reportgenerator.py
```

O script vai gerar um arquivo `.html` que você pode abrir direto no navegador.
